# Samaa Elbakry Portfolio

A modern and interactive **frontend portfolio** built with Next.js, showcasing my projects, skills, and experience with a strong focus on performance, animations, and user experience.

---

## Tech Stack

This project is built using modern frontend technologies:

* **Next.js** – for building fast and optimized web applications (App Router & SSR) ([GitHub][1])
* **React.js** – component-based UI development
* **TypeScript** – for type safety and better scalability
* **Tailwind CSS** – for responsive and modern styling
* **GSAP (GreenSock)** – for smooth and advanced animations
* **Three.js / 3D Elements** – for interactive 3D visuals
* **Vercel** – for deployment

---

##  Features

*  Modern and clean UI design
*  Fully responsive across all devices
*  High performance with Next.js optimization
*  Smooth animations using GSAP
*  Interactive 3D elements
*  Projects showcase section
*  Contact section
